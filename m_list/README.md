## 概要

M List は、SketchUp モデル内の Group と Component を自動で検出し、見積もり用のリストを作成します。

- **自動リスト取得**: モデル内の Group/Component を階層構造で一覧化
- **形状の自動判定**: 棒状・面状・入れ子・自由形状などを自動判定
- **材料割り振り**: 角材・板材・仕上材をドロップダウンで選択
- **価格の自動計算**: 寸法と素材から価格を自動計算
- **再計算・検証**: リスト全体の価格を再計算し、仕上材の反映漏れ・親子の不整合などを検出して警告
- **集計表示**: 選択項目の必要素材数・余り・合計金額を表示
- **CSV 入出力**: 見積もりデータの保存・読み込み

## 必要な環境

- SketchUp 2017 以降（persistent_id を使用）
- Ruby 2.2 以降（SketchUp に同梱）

## インストール

1. `m_list.rb` を SketchUp の Plugins フォルダに配置
2. `m_list` フォルダを同じ Plugins フォルダ内に配置
3. SketchUp を再起動

**フォルダ構成の例:**
```
Plugins/
├── m_list.rb
└── m_list/
    ├── panel.html
    ├── materials_beam.csv
    ├── materials_board.csv
    ├── materials_finish.csv
    ├── icons/              （任意・形状アイコン用）
    │   ├── icon_nested.png
    │   ├── icon_beam.png
    │   ├── icon_board.png
    │   ├── icon_freeform.png
    │   ├── icon_other.png
    │   └── icon_calculator.png
    └── README.md
```

※ icons フォルダがなくても動作します。形状アイコンが表示されない場合は icons フォルダを追加してください。

## 起動方法

1. SketchUp のメニューから **拡張機能 → M List** を選択
2. パネルが表示されます

## オンラインアップデート

M List は、ダウンロード済みユーザーがオンラインで最新バージョンを確認できる機能を搭載しています。

### ユーザー向け

- **パネルの「更新確認」ボタン**: クリックで最新バージョンを確認
- **拡張機能 → M List - アップデートを確認**: メニューからも確認可能
- パネル表示時に **2週間ごとに自動チェック**（配布者がバージョンファイルをホスティングしている場合）

### 配布者・開発者向け（バージョンファイルのホスティング）

オンラインアップデートを有効にするには、`m_list.rb` の以下の定数を設定し、バージョン情報ファイルをホスティングしてください。

1. **`UPDATE_CHECK_URL`**: バージョン情報を返す URL（JSON またはプレーンテキスト）
2. **`UPDATE_DOWNLOAD_URL`**: ダウンロードページの URL（フォールバック用）

**バージョンファイルの形式（JSON）:**
```json
{
  "version": "1.1.3",
  "download_url": "https://example.com/m_list/download",
  "release_notes": "更新内容の説明"
}
```

**プレーンテキスト形式:** バージョン番号のみ（例: `1.1.3`）

サンプルは `m_list/version.json.sample` を参照してください。GitHub の raw URL、自前サーバー、クラウドストレージなどでホスティング可能です。

## 基本的な使い方

### 1. リストの取得

1. モデルに Group または Component を作成
2. パネルの **「同期」** ボタンをクリック
3. モデル内の Group/Component が階層構造でリストに表示されます

### 2. 材料の割り当て

- **棒状** の部品: 角材・丸太など棒状素材をドロップダウンで選択
- **面状** の部品: 板材を下地材、塗料などを仕上材として選択
- **その他**: 素材名を直接入力

### 3. 価格の計算

- 棒状・面状で素材を選択すると、寸法から価格が自動計算されます
- 計算値に戻したい場合は **計算アイコン** をクリック
- 手動で価格を入力することも可能です

### 4. 保存

- **保存**: プロジェクトフォルダに .skp と CSV を保存（新規の場合はフォルダ選択）
- **別名保存**: 別のフォルダに保存
- **CSVのみ保存**: 見積もり CSV のみ保存

## パネル機能の説明

### ツールバー

| ボタン | 機能 |
|--------|------|
| **同期** | モデルから Group/Component を再取得し、リストを更新 |
| **保存** | プロジェクトを保存（.skp + CSV） |
| **保存 ▼** | 別名保存 / CSVのみ保存 |
| **再計算** | 価格を材料・仕上材から再計算し、親を子の合計に合わせる。不整合を検出して警告 |
| **その他 ▼** | CSV読み込み / 素材CSV更新 / 更新確認 |

### リスト操作

- **クリック**: 項目を選択し、3D ビューで該当オブジェクトにズーム
- **Ctrl/Cmd + クリック**: 複数選択
- **名前・価格・メモ・素材欄**: どこをクリックしても選択・集計が表示されます
- **展開ボタン** (▶): 入れ子の子項目を表示/非表示
- **形状アイコン**: 棒状・面状など形状を上書き（入れ子は変更不可）
- **計算アイコン**: 価格を自動計算値に戻す（棒・面状のみ）
- **表示アイコン** (👁): 見積もりから外す（非表示）/ 表示に戻す

### 選択項目の集計

リストで項目を選択すると、下部に以下が表示されます。

- **最終サイズ・面積**: 棒の本数・総延長、面状の枚数・片面合計面積
- **棒状 必要個数**: 各角材の必要本数と余り
- **面状 必要個数**: 各板材の必要枚数と余り
- **仕上材 必要個数**: 塗料などの必要個数と余り
- **合計金額**: 選択項目の合計

---

## 形状タイプについて

部品は寸法から自動で以下のように判定されます。

| 形状 | 説明 |
|------|------|
| **入れ子** | 子の Group/Component を持つ |
| **棒** | 細長い角材・丸太形状 |
| **面状** | 板状（厚さ 60mm 以下など） |
| **自由形状** | 6面以上の複雑な形状 |
| **その他** | 上記に該当しない形状 |

棒状・面状以外は形状を手動で上書きできます（入れ子は構造のため変更不可）。

## 素材CSVの設定

### ファイルの場所

- **棒状**: `materials_beam.csv`
- **面状**: `materials_board.csv`
- **仕上材**: `materials_finish.csv`

プロジェクト保存時に、プロジェクトフォルダに素材 CSV がコピーされます。  
「素材CSV更新」でプロジェクトフォルダ内の CSV を再読み込みできます。

### 棒状 (materials_beam.csv)

| 列 | 説明 |
|----|------|
| material_id | 素材ID（一意） |
| name | 表示名 |
| unit_size | 規格サイズ（例: 30x40x3000 = 30×40×3000mm） |
| unit_price | 単価（円） |
| loss_rate | ロス率（%） |

### 面状 (materials_board.csv)

| 列 | 説明 |
|----|------|
| material_id | 素材ID |
| name | 表示名 |
| thickness_mm | 厚み（mm） |
| unit_size | 規格サイズ（例: 910x1820 = 910×1820mm） |
| unit_price | 単価（円） |
| loss_rate | ロス率（%） |

### 仕上材 (materials_finish.csv)

| 列 | 説明 |
|----|------|
| finish_id | 仕上材ID |
| name | 表示名 |
| unit_size | 単位サイズ |
| unit_price | 単価 |
| m2_coverage_per_unit | 1単位あたりの塗布面積（m²） |
| 1m2_price | 1m²あたりの追加単価（板材に貼る場合など） |
| loss_rate | ロス率（%） |

## プロジェクト保存の仕組み

**保存** を実行すると、次のフォルダ構成で保存されます。

```
プロジェクトフォルダ/
├── プロジェクト名.skp      # SketchUp モデル
├── プロジェクト名_estimate.csv  # 見積もりデータ
├── materials_beam.csv      # 棒状素材
├── materials_board.csv    # 面状素材
└── materials_finish.csv   # 仕上材
```

初回保存時はフォルダを選択するダイアログが表示されます。

## リストと3Dの連動

- **リスト → 3D**: リストの項目をクリックすると、3D ビューで該当オブジェクトが選択され、ズームされます
- **3D → リスト**: 3D ビューで Group/Component を選択すると、リストの該当行がハイライトされます
- **入れ子**: 入れ子の子項目を選択した場合も、親が自動で展開され、対応するオブジェクトが表示されます

## トラブルシューティング

### リストが空のまま

- モデルに Group または Component が存在するか確認してください
- **同期** ボタンをクリックしてリストを再取得してください

### ズームが正しく動かない

- **同期** ボタンでリストを再取得してください（Undo 後などで ID が変わることがあります）
- 該当オブジェクトが見つからない場合は、パネルにメッセージが表示されます

### 価格が材料・仕上材と合っていない

- **再計算** ボタンをクリックすると、材料・仕上材から価格を再計算し、親は子の合計に合わせます
- 仕上材を選んだのに値段に反映されていない、親の値段が子の合計と違うなどの不整合があれば警告されます

### 集計がおかしい

- 入れ子の親を選択すると、子孫の素材も含めて集計されます
- 同じコンポーネントの複数インスタンスも正しく合算されます
- 素材が未割り当ての部品は集計に含まれません

### 素材CSVが読み込まれない

- プロジェクトフォルダ内に `materials_*.csv` があるか確認してください
- 先に **プロジェクト保存** を行うと、デフォルトの素材 CSV がコピーされます
- **素材CSV更新** で再読み込みできます

### CSV の文字化け

- UTF-8 で保存された CSV を使用してください
