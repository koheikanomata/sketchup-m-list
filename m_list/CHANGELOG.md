# Changelog

## [1.1.5] - 2025-03-01

### 追加

- **ダイナミックコンポーネント対応**:
  - `panel_thickness` 属性（cm）を板厚として優先読み込み
  - bounds 変化時の自動再計算（板厚変更後、同期で価格を更新）
  - Hidden エンティティをリスト・集計から除外（Show_Legs 等の非表示部品を価格に含めない）

## [1.1.4] - 2025-03-01

### 追加

- **自動更新**: アップデート確認時に「今すぐ自動更新」を選択すると、SketchUp 内でプラグインファイルをダウンロードして上書き。完了後、SketchUp を再起動するだけで反映

## [1.1.3] - 2025-03-01

### 変更

- **ツールバー整理**: パネル上部のボタンを整理し、「CSV読み込み」「素材CSV更新」「更新確認」を「その他」ドロップダウンに集約。メイン操作（同期・保存・再計算）を前面に配置

## [1.1.2] - 2025-02-26

### 追加

- **オンラインアップデート機能**: ダウンロード済みユーザーがオンラインで最新バージョンを確認できるように
  - パネルに「更新確認」ボタンを追加
  - 拡張機能メニューに「M List - アップデートを確認」を追加
  - パネル表示時に2週間ごとに自動でバージョンチェック（要: バージョンファイルのホスティング）
  - `m_list.rb` の `UPDATE_CHECK_URL` と `UPDATE_DOWNLOAD_URL` を設定すると有効化

## [1.1.1] - 2025-02-15

### 修正

- 名前・値段・メモなどの入力欄をクリックした際、ズーム機能と競合して入力カーソルが外れる問題を修正。入力欄クリック時はズームを実行しないように変更

## [1.1.0] - 2025-02-15

### 追加

- **再計算** ボタン: ツールバーに追加。リスト全体の価格を材料・仕上材から再計算し、親を子の合計に合わせる
- 検証・警告: 仕上材の反映漏れ、親子の値段不整合、素材CSV未登録、オブジェクト未検出などを検出して警告表示

## [1.0.0] - 2025-02-15

### 初回リリース

- 3Dモデルからの自動リスト取得（Group/Component を階層構造で一覧化）
- 形状の自動判定（棒状・面状・入れ子・自由形状・その他）
- 棒材・板材・仕上材の材料割り振り（プルダウン選択）
- 棒材：断面サイズによる自動マッチング
- 板材：厚み・下地材・仕上材の3プルダウン、厚みによる下地材の絞り込み
- 板材：1220x2440 規格の優先
- 価格の自動計算
- 必要素材数の集計表示（素材名表示）
- CSV 入出力
- プロジェクト保存（.skp + CSV + 素材CSV）
- リストと3D選択の連動（ズーム・ハイライト）
